🧠### Content-DNA-Engine: Elite Script Intelligence & Competitive Analytics

Content-DNA-Engine is a machine learning and natural language processing system designed to analyze YouTube video scripts and channel trajectories through semantic structure extraction and competitive benchmarking.

It bridges the gap between competitive intelligence and creative content strategy, enabling creators to systematically evaluate narrative performance, retention behavior, and packaging effectiveness against top-performing market benchmarks.

🛠️ Technical Stack
Artificial Intelligence: Gemini Pro API Layer (gemini-2.5-flash with strict JSON schema enforcement)
Data Aggregation: YouTube Data API v3, YouTube Transcript API
Web Interface: Streamlit (custom glassmorphic UI theme system)
Languages & Tooling: Python 3.9+, Python-dotenv, JSON
